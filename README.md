# Interoperability Project

The CST Interoperability Project is an initiative of the Lyrasis Organizational Home, supported by the Lyrasis CST Growth Fund, facilitated by Jessica Farrell of Redstart Works consulting.

Under the guidance of the Organizational Home and in close consultation with the communities, this project is designed to equitably support each of these important technologies, and center community, sustainability, and interoperability.

This repository will be used to collaborate on the work products.

The first phase of the project is to define functional requirements for interoperability workflows. 

# How to Participate
The Lyrasis CST Interoperability Project needs your feedback to refine interoperability solutions!

Here is an overview of the feedback process:

1. Ideas are developed in the workshops (facilitated Dec. 10-11, 2025) and via the [survey](https://lyrasisorghome.github.io/InteroperabilityProject/feedback/) (closes Jan. 15, 2026).
2. The CST Interoperability consultant (Jess) posts ideas to (1) [GitHub Issues](https://github.com/lyrasisorghome/InteroperabilityProject/issues) and (2) [the Interoperability Google Group](https://groups.google.com/a/lyrasislists.org/g/cst-interoperability). 
	- Community members and stakeholders may also post ideas. Go to Issues > Issue > New Issue and use the Integration Idea template. It's okay if the idea is not exactly an integration, still use this template to log any kind of interoperability solution. You can leave it up to Jess to set the right labels, project, and milestones after an Issue is created (data on the right pane).
3. All stakeholders are invited to discuss the ideas by commenting on the GitHub issue or responding to the Google Group thread about the idea. The goal is to turn vague ideas into interoperability solutions by discussing how you would use the solution to enhance your workflows.
    - See below for a list of links to Issues and the Project filtered to each community.
    - If conversations about one idea happen in both GitHub and Google Groups, the project team will facilitate between the two spaces to keep everyone on the same page.
5. Jess (and anyone else who wants to jump in!) will work with community members to expand on ideas as they attract enough detail to be considered for development.
6. The [Interoperability Idea Generation, Expansion, and Prioritization Project](https://github.com/orgs/lyrasisorghome/projects/1/views/1) will show ideas moving through the refinement process. To see just your community's ideas, filter by community label on the left, or use the tabs across the top.
